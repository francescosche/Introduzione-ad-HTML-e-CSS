Introduzione a jQuery

← Torna al menù

jQuery è un framework JavaScript che facilita la realizzazione di script per l'interazione con l'utente.

Questa struttura permette infatti, prima di tutto, di utilizzare i selettori CSS per identificare gli elementi su cui agire. Infatti, invece di utilizzare la lunga sintassi del document.getElementByQualcosa, è possibile utilizzare un semplice $, a cui dovrà essere passata come parametro una stringa che identifica il selettore CSS corrispondente all'elemento.

Vediamo un esempio: 

document.getElementById('topbar');
        // equivale a scrivere
    $('#topbar');

    document.getElementsByTagName('input');
        // equivale a scrivere
    $('input');

    document.getElementsByClassName('titoli');
        // equivale a scrivere
    $('.titoli');

La stessa cosa succede anche per i metodi da richiamare sugli elementi che, oltre ad essere molto più brevi, sono spesso anche più comprensibili e comprendono più segnature per lo stesso metodo. Ovvero, richiamando (ad esempio) e.html(); si ottiene l'HTML all'interno dell'elemento e, mentre richiamando e.html("<b>Ciao!</b>");, si setta l'HTML all'interno dell'elemento e in modo che contenga <b>Ciao!</b>, utilizzando così un solo metodo per eseguire due operazioni differenti.
