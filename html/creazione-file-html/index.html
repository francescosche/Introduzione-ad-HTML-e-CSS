Per "iniziare" realmente a scrivere codice HTML, bisogna prima di tutto sapere, appunto, dove questo codice deve essere scritto.

A corto di risorse o se non si ha voglia di scaricare sfotware aggiuntivi, si può tranquillamente utilizzare un editor di testo di quelli già presenti sui computer (come Blocco Note, TextEdit, gedit, ecc.).

Se invece si vuole scaricare un software apposito per la scrittura di questo codice, quello che io consiglio è Atom.io, l'editor che sto utilizzando in questo momento per scrivere questa pagina.

L'unica cosa importante che bisogna ricordare, qualsiasi sia l'editor che vogliate scegliere per scrivere il vostro sorgente, è quella di salvare il file che scriverete con estensione .html, ovvero, dall'editor, scegliere File > Salva con nome.. > nomedelfile.html
