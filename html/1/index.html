Il tag <html>

← Torna al menù

Il tag html è il più importante dei file HTML, appunto perchè definisce l'inizio e la fine di un documento.
Per far interpretare un file in HTML5, inoltre, è necessario inserire all'inizio del file il codice

<!DOCTYPE html> Questo serve a far capire al browser con quale versione del linguaggio interpretare il file, dato che per HTML4.01 veniva usato <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd"> Il tag html deve contenere tutto il contenuto della pagina (visibile e non), quindi dovrò essere aperto e chiuso, ovvero: <html> ... </html>

I commenti

In qualsiasi parte del codice, come vedremo, è possibile inserire dei commenti per comprendere meglio il codice che si sta scrivendo, in modo da non dimentarsi il motivo per cui sono fatte certe decisioni.
Un commento può essere inserito semplicemente seguendo una sintassi del genere:

<!-- Premi il link qui sotto per continuare -->
