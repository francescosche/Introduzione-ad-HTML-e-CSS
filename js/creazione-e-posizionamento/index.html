Creazione di file e posizionamento

← Torna al menù

Come per il CSS, la creazione di file JavaScript è molto semplice: basta infatti scrivere un documento di testo contenente il codice JS e salvarlo con estensione .js

Se invece si vuole implementare il codice direttamente all'interno del file HTML, è possibile inserirlo nell'<head> tramite il tag <script>...</script> che conterrà il relativo codice JavaScript da utilizzare.

Un codice come il seguente: 

<!DOCTYPE html>
<html>
    <head>
        <script>
            window.onload = function(e) {
                alert("Hello world!");
            }
        </script>
    </head>
    <body>
        Hello world di prova.
    </body>
</html>

Genererà un codice come quello in questa pagina (provalo!)

Allo stesso modo, un codice come il seguente: 

// file s1.js
window.onload = function(e) {
    alert("Hello world!");
}

<!-- file ex2.html -->
<!DOCTYPE html>
<html>
    <head>
        <script src="s1.js"></script>
    </head>
    <body>
        Hello world di prova.
    </body>
</html>

Genererà un codice come quello in questa pagina (provalo!)
