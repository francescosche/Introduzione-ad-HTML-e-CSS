Introduzione a JavaScript

← Torna al menù

JavaScript è un linguaggio di programmazione che permette di modificare in tempo reale le pagine HTML.

Come abbiamo visto, infatti, il codice HTML, una volta scritto, non può essere modificato. È possibile utilizzare animazioni in CSS o modificare i dati memorizzati in un browser tramite il localStorage, ma non è possibile, in HTML, modificare (da lato client) il codice HTML della pagina stessa.

Inizialmente introdotto come DHTML (Dynamic HTML) e poi trasformato in JavaScript, questo linguaggio permette di utilizzare vari metodi per l'elaborazione e la modifica di dati, che possono interagire in tempo reale con la pagina HTML sulla quale vengono richiamati, modificandola di conseguenza.
