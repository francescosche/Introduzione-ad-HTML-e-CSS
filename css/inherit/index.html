Il valore inherit

← Torna al menù

Il valore inherit ha un ruolo speciale nel CSS.
Questo valore, infatti, specifica che il valore da dare a quell'attributo, non verrà specificato, ma dovrà essere richiesto all'elemento padre dell'elemento stesso.

Ad esempio, se in una div aggiungiamo un tag span che contiene del testo, quando andremo a specificare font-family: inherit; per il tag span, quest'ultimo capirù che la font-family è specificata all'interno del codice CSS riferito alla div.

Questo valore viene usato molto spesso nei css di default dei browser, per fare in modo che degli attributi vengano specificati nel CSS del body e si riflettano così all'interno di tutta la pagina.
Questo valore ha infatti un effetto "a cascata", qualora due elemento consecutivi dovessero applicarlo.
