Tipi di posizionamento e commenti in CSS

← Torna al menù

Il codice CSS di una pagina può essere posizionato in parti diverse, in base al ruolo che deve svolgere.
L'opzione più sbrigativa è quella di inserire, come attributo di un elemento, l'attributo style e scriverci come valore il codice CSS che dovrà agire su quell'elemento.

<span style="color: blue;">Testo blu</span>

Una seconda opzione è quella di inserire il codice CSS scritto in maniera corretta all'interno del tag style, il quale deve essere contenuto nell'head.

<head> <style> span {color: blue;} </style> </head>

La terza opzione, quella che io raccomando a tutti, è quella di scrivere tutto il codice CSS in maniera corretta all'interno di un altro file, con estensione .css, e integrarlo poi all'interno della pagina HTML tramite il seguente tag, contenuto nell'head.

<head> <link href="foglio.css" rel="stylesheet" type="text/css" /> </head>

Ovviamente, il nome del foglio di stile da implementare dovrà essere rimpiazzato con quello scritto da voi
