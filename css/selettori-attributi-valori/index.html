Selettori, attributi e valori

← Torna al menù

Le uniche 3 cose da sapere riguardo il CSS sono: selettori, attributi e valori.
Si, sono poche cose, ma le combinazioni fra esse sono praticamente infinite, soprattutto se si considera che a ciascun elemento HTML possono essere applicati più attributi, e, di conseguenza, più valori.

Vediamo meglio di cosa stiamo parlando.

I selettori

Quando si deve scrivere un codice CSS, bisogna prima di tutto conoscere l'elemento (o gli elementi) che si vogliono andare a modificare.
Per selezionatre questo/i elemento/i, si usano, appunto, i selettori.

Un selettore può essere di 3 tipologie:

  • Per tag
  • Per classe
  • Per identificativo
Un selettore per tag, come si può capire, andrà a modificare tutti i tag che corrispondono a quel tipo.
Un selettore per classe, invece, selezionera tutti gli elementi HTML a cui sarà stato assegnato l'attributo class uguale a quello specificato.
I selettori per id, selezioneranno invece tutti gli elementi HTML con l'attributo id uguale a quello specficato.

Visti così, però, sembra non ci siano molte differenze fra i selettori.
La grande differenza si ha, però, nel numero di elementi che ciascuno di essi andrà a modificare. Infatti, usando un selettore di tag, si andranno a modificare tutti gli elementi corrispondenti a quel tag
Usando invece un selettore per classe, si andranno a modifica solo i tag che avranno l'attributo class corrispondente a quello (che potrebbe essere attribuito anche a tag differenti).
Usando invece un selettore di id, si andranno a modificare solo gli elementi che hanno l'attributo id uguale a quello specificato.
In questo caso, però, l'attributo id viene solitamente usato per identificare un unico elemento nella pagina, poichè è uno dei metodi più semplici per selezionare un elemento in JavaScript.

Per questo motivo, consiglio sempre di utilizzare le classi per dare lo stile agli elementi (anche se è uno solo) e gli identificatori per utilizzare gli elementi in JavaScript.

Dal lato pratico, invece, per specificare quale tipo di selettore si vuole usare, bisogna far precedere alla stringa della selezione un carattere:

  • nessuno — Per utilizzare un selettore di tag
  • . - Per utilizzare un selettore di classe
  • # - Per utilizzare un selettore di id
Vediamoli però in maniera più "diretta":

<style> strong { color: red; } .classe { color: blue; } #identificativo { color: green; } </style> <strong>Questo testo dovrebbe essere rosso (e grassetto)</strong><br /> <span class="classe">Questo testo dovrebbe essere blu</span><br /> <span id="identificativo">Questo testo dovrebbe essere verde</span>

Questo codice, dovrebbe generare il seguente risultato:

Questo testo dovrebbe essere rosso (e grassetto)
Questo testo dovrebbe essere blu
Questo testo dovrebbe essere verde

Come possiamo vedere, il primo selettore modifica solamente il tag strong, il secondo modifica solo il tag span con attributo class=classe e il terzo modifica solamente il tag span con attributo id=identificativo.

Gli attributi

Nell'esempio precedente, implicitamente, abbiamo usato un selettore per modificare il colore dei vari elementi.

Il selettore, nel nostro caso, era, appunto, color.

In CSS esistono centinaia (se non migliaia di attributi), che però vedremo meglio andando avanti nello studio del CSS.

I valori

Per quanto riguarda i valori, possono essere di molti tipi, ma quelli che si ripetono più spesso sono solitamente quelli riguardo le dimensioni e i colori.

Riguardo le dimensioni, è possibile impostarle utilizzando varie unità di misura:

  • px — Specificando la dimensione in pixel
  • in - Specificando la dimensioni in inch
  • % - Specificando la dimensione in percentuale rispetto all'elemento "contenitore" più vicino
  • vw - Specificando la dimensione in relazione diretta alla larghezza della finestra (cambierà quando la finestra verrà ridimensionata)
  • vh - Specificando la dimensione in relazione diretta all'altezza della finestra (cambierà quando la finestra verrà ridimensionata)

Riguardo i colori, invece, ci sono diverse tipologie di rappresentazione:

  • rgb() — Specificando, all'interno delle parentesi, in ordine, la quantità (fra 0 e 255) di red, green e blue
  • rgba() - Come il precedente, con l'unica differenza che va specficato un quarto valore: alpha, che rappresenta la percentuale di opacità del colore (fra 0.00 e 1.00)
  • #HEX - Con questa rappresentazione, bisogna utilizzare 6 caratteri, suddivisi in 3 coppie, ciascuna delle quali rappresenta (in esadecimale, con un valore compreso fra 00 e FF), la quantità, in ordine, di red, green e blue
  • stringa — Alcune stringhe sono riconosciute dal browser come colori, ma si perdono molte gradazioni (elenco completo qui)

Volendo ora riprendere l'esempio precedente, modificato con le nuove conoscenze, possiamo scrivere il codice:

<style> strong { color: rgba(255,0,0); } .classe { color: #0000ff; } #identificativo { color: green; } </style> <strong>Questo testo dovrebbe essere rosso (e grassetto)</strong><br /> <span class="classe">Questo testo dovrebbe essere blu</span><br /> <span id="identificativo">Questo testo dovrebbe essere verde</span>

Che rappresenterà:

Questo testo dovrebbe essere rosso (e grassetto)
Questo testo dovrebbe essere blu
Questo testo dovrebbe essere verde
